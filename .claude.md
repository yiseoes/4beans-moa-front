# MOA 프론트엔드 테마 시스템 가이드

## 테마 개요

MOA 프론트엔드는 6가지 테마를 지원합니다:
- **default** - 기본 슬레이트 스타일
- **classic** - 클래식 화이트 + 퍼플 악센트
- **dark** - 다크 모드
- **pop** - 네온/팝 스타일 (두꺼운 검정 테두리, 핑크/시안/라임 색상)
- **portrait** - 파스텔 그라데이션 스타일
- **christmas** - 크리스마스 테마 (빨강/초록 색상, 눈 내리는 배경)

---

## 테마 스토어

**파일:** `src/store/themeStore.js`

```javascript
import { useThemeStore } from "@/store/themeStore";

// 사용법
const { theme, setTheme } = useThemeStore();
```

**기본 테마:** `pop`

---

## 컴포넌트별 테마 스타일 패턴

### 기본 패턴

```jsx
import { useThemeStore } from "@/store/themeStore";

// 테마별 스타일 정의
const componentThemeStyles = {
  default: {
    // default/pop 테마 스타일
    border: "border-2 border-black",
    button: "bg-indigo-600 hover:bg-indigo-700",
    hover: "hover:bg-slate-50",
  },
  christmas: {
    // 크리스마스 테마 스타일
    border: "border border-gray-200",
    button: "bg-red-800 hover:bg-red-900",
    hover: "hover:bg-red-50",
  },
};

// 컴포넌트에서 사용
export function MyComponent() {
  const { theme } = useThemeStore();
  const themeStyle = componentThemeStyles[theme] || componentThemeStyles.default;

  return (
    <div className={themeStyle.border}>
      <button className={themeStyle.button}>버튼</button>
    </div>
  );
}
```

## 주요 테마 설정 파일

### themeConfig.jsx
**경로:** `src/config/themeConfig.jsx`

전역 테마 설정:
- `themeConfig` - 페이지 테마 설정
- `headerThemes` - 헤더 테마 설정
- `navPillStyles` - 네비게이션 pill 스타일
- `ThemeBackground` - 테마별 배경 컴포넌트
- `ThemeSwitcher` - 테마 변경 UI 컴포넌트

### 배경 컴포넌트
- `AnimatedGradient` - Classic 테마 그라데이션
- `DarkGradient` - Dark 테마 그라데이션
- `PortraitBackground` - Portrait 테마 파스텔 배경
- `ChristmasBackground` - Christmas 테마 눈 내리는 배경
- `GridPattern` - 그리드 패턴 오버레이

## 새 컴포넌트에 테마 추가하기

### 1. 테마 스타일 객체 정의
```jsx
const myComponentThemeStyles = {
  default: {
    // Pop/기본 스타일
    primaryButton: "bg-indigo-600 hover:bg-indigo-700",
    border: "border-2 border-black",
    hover: "hover:bg-slate-50",
  },
  christmas: {
    // 크리스마스 스타일
    primaryButton: "bg-red-800 hover:bg-red-900",
    border: "border border-gray-200",
    hover: "hover:bg-red-50",
  },
};
```

### 2. 훅과 스타일 적용
```jsx
export function MyComponent() {
  const { theme } = useThemeStore();
  const themeStyle = myComponentThemeStyles[theme] || myComponentThemeStyles.default;

  return (
    <div className={themeStyle.border}>
      <button className={`${themeStyle.primaryButton} text-white`}>
        버튼
      </button>
    </div>
  );
}
```

### 3. 체크리스트
- [ ] `border-black`, `border-slate-900` → 테마별 분리
- [ ] `bg-indigo-*` → 테마별 분리
- [ ] `shadow-2xl`, 검정 쉐도우 → 테마별 분리
- [ ] `hover:bg-slate-*` → 테마별 분리
- [ ] `bg-pink-*`, `bg-cyan-*`, `bg-lime-*` → 테마별 분리

## 검색 패턴 (팝 요소 찾기)

팝 스타일 요소를 찾을 때 사용하는 grep 패턴:
```bash
# 검정 테두리
border-black|border-slate-900|border-b-4 border-black

# 검정 쉐도우
shadow-2xl|shadow-\[4px_4px_0px

# indigo 색상
bg-indigo-|text-indigo-|hover:bg-indigo-

# 팝 악센트 색상
bg-pink-|bg-cyan-|bg-lime-

# 하드코딩된 hover
hover:bg-slate-50|hover:bg-slate-100
```

## 주의사항

1. **테마 스타일은 항상 객체 안에** - 하드코딩된 className 대신 테마 스타일 객체 사용
2. **default 폴백** - `themeStyles[theme] || themeStyles.default` 패턴 사용
3. **hover 색상도 테마화** - `hover:bg-slate-50` → 테마별 hover 색상
4. **일관된 명명** - `themeStyle.border`, `themeStyle.hover`, `themeStyle.primaryButton` 등
5. **Neo 컴포넌트** - `NeoButton`, `NeoCard` 등은 이미 테마 대응됨
